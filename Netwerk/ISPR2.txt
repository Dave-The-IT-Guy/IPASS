Basis instellingen:
- sudo passwd
- configure
- set system host-name {hostname}
- set system login user vyos authentication plaintext-password {password}
- set system login user admin authentication plaintext-password {password}
- set interfaces ethernet eth5 address dhcp
- set interfaces ethernet eth5 description internet
- set service ssh
- commit
- save
- exit
- ip a (voor het bekijken van het IP)

Open een ssh sessie en plak de volgende regels in de terminal:

configure
set interfaces ethernet eth0 address 10.58.2.90/30
set interfaces ethernet eth0 description ISPR2.1_ZR2.3
set interfaces ethernet eth1 address 10.58.2.82/30
set interfaces ethernet eth1 description ISPR2.2_ZR1.4
set interfaces ethernet eth2 address 10.58.2.45/30
set interfaces ethernet eth2 description ISPR2.3_ISPR4.1
set interfaces ethernet eth3 address 10.58.2.49/30
set interfaces ethernet eth3 description ISPR2.4_ISPR3.2
set interfaces ethernet eth4 address 10.58.2.42/30
set interfaces ethernet eth4 description ISPR2.5_ISPR1.5
commit
save
exit
