Basis instellingen:
- sudo passwd
- configure
- set system host-name {hostname}
- set system login user vyos authentication plaintext-password {password}
- set system login user admin authentication plaintext-password {password}
- set interfaces ethernet eth5 address dhcp
- set interfaces ethernet eth5 description internet
- set service ssh
- commit
- save
- exit
- ip a (voor het bekijken van het IP)

Open een ssh sessie en plak de volgende regels in de terminal:

configure
set interfaces ethernet eth0 address 10.58.2.46/30
set interfaces ethernet eth0 description ISPR4.1_ISPR2.3
set interfaces ethernet eth1 address 10.58.2.38/30
set interfaces ethernet eth1 description ISPR4.2_ISPR1.4
set interfaces ethernet eth2 address 10.58.2.65/30
set interfaces ethernet eth2 description ISPR4.3_UR2.1
set interfaces ethernet eth3 address 10.58.2.69/30
set interfaces ethernet eth3 description ISPR4.4_UR1.2
set interfaces ethernet eth4 address 10.58.2.62/30
set interfaces ethernet eth4 description ISPR4.5_ISPR3.5
commit
save
exit
