Basis instellingen:
- sudo passwd
- configure
- set system host-name ISPR3
- set system login user vyos authentication plaintext-password {password}
- set system login user admin authentication plaintext-password {password}
- set interfaces ethernet eth5 address dhcp
- set interfaces ethernet eth5 description internet
- set service ssh
- commit
- save
- exit
- ip a (voor het bekijken van het IP)

Open een ssh sessie en plak de volgende regels in de terminal:

configure
set interfaces ethernet eth0 address 10.58.2.34/30
set interfaces ethernet eth0 description ISPR3.1_ISPR1.3
set interfaces ethernet eth1 address 10.58.2.50/30
set interfaces ethernet eth1 description ISPR3.2_ISPR2.4
set interfaces ethernet eth2 address 10.58.2.53/30
set interfaces ethernet eth2 description ISPR3.3_UR1.1
set interfaces ethernet eth3 address 10.58.2.57/30
set interfaces ethernet eth3 description ISPR3.4_UR2.2
set interfaces ethernet eth4 address 10.58.2.61/30
set interfaces ethernet eth4 description ISPR3.5_ISPR4.5
commit
save
exit
